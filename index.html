回家的旅途 第一章

发表于 2022-07-26 更新于 2023-04-11 分类于 小说 Waline： 本文字数： 790 阅读时长 ≈ 1 分钟

萧澪哈了一口气，搓了搓指尖透红的手，回头问表弟：”冷吗？坚持一下车马上来了。“

表弟点点头，白色的雾气穿过青涩的脸颊。

周围等车的寥寥数人，是啊，在这个大家团聚的日子，现在还坐公共交通的还有几人？

萧澪松了松背包的肩带，包里不过简单几件衣服，但背着时间久了，肩膀发酸，不自觉的想要弯下来。

车来了，司机专注着看着前方，冷漠的像个机器人。

萧澪刷了卡，进入了这个没有冷风的铁皮盒子。他快速扫了车厢情况，只有一名老者坐在后排。

萧澪回头看了下表弟，确保他跟在身后，这个空挡其他乘客已经在前排落座，萧澪随着车子的启动摇晃了下，小心走向靠近老者的座位。

老者一身灰色长风衣，里面是黑色圆领毛衣配格子衬衫，脚上一双雪地靴，身旁放着一个背包，手指轻轻击打着自己大腿，眼睛看着上车的位置。

萧澪坐在座位上，开始想着接下来的打算与行程安排，在脑海中预演几遍，眼睛忍不住开始打量周围的乘客，前排乘客在等车的时候已经看好几遍了，萧澪于是眼神又回到老者身上。

老者也不以为意，还是看着上车的地方。

萧澪没法使用手机，毕竟天气冷，而且接下来好几个地方还需要手机确认，不能浪费电量。他略感无聊。

萧澪向老者搭话：”你好，今天回家和亲人团聚吗？“

老者转头看向萧澪：”这里的天气有点冷，这是我的旅途。“

萧澪没想到这种回应，有些出乎意料，他接着问：”您什么时候开始的旅行？“

老者看向窗外，低沉的声音包含着回忆：“我有些记不清了”

萧澪觉得这个路途上碰到了有趣的人，他问道：“那您去过什么地方啊”

老者看向他：“如果你想了解，可以来我暂住的地方，坐着聊聊。”

萧澪有些警惕起来，问道：”你住在哪里呢？“

”一家旅馆，马上到了。“

萧澪刚在犹豫的时候，表弟轻轻碰了他下，示意他看看背包，背包的一角露出金色。

老者似乎并不在意，又说道：”我很久没怎么和人聊过天了，如果你有空听听，我不防给你些钱，当作你时间的补偿。“

车到站了。萧澪和表弟跟着老者下车了。

当下的时光

距离新冠疫情爆发已经近两年，疫苗没有成为“避魔圈”，层出不穷的变异体继续肆虐，伴随着美国接近第一次高峰的确诊数量，学生开始返回校园。

与我一同来的同学此时硕士毕业，在回国前夕说好再聚一次，想想距离上一次见面，大概是在刚来的时候吧，偶尔微信联系，谈论的内容回忆起我们还在同一个城市。

那天是个晴天，温度很高，转了一趟地铁，接近一个小时的路程，地铁上除了报站的声音，多了提醒戴上口罩。因为是周末的缘故，乘客并不多，当然现在还没到九月，回想曾经的早高峰和晚高峰往往有位置站着，已经心满意足了。

下了地铁，跟随导航兜了几圈，终是到了实验室的楼下。他没什么变化，头发长了点，稍微圆润了点，当然这是在写毕业论文时常见的现象，我跟随他去游览学校。

青春的活力洋溢在校园中，街边常常有长跑爱好者匆匆而过，潮湿的面孔，轻盈的身影，有节奏的随整个城市继续跳动。校园附近的餐厅，人们在门口留下电话等待进入，一如往常在餐桌上高谈阔论增进感情，只是在呈上饭菜之时多了一份仪式，轻轻脱下口罩，搓搓免洗洗手液。

在吃饭前夕，他又跑去进行了下实验，这份热情是我远远不及的，问及毕业后的打算，已经在上海找到了工作，距离他南京的家人也比较近，想来上海这座城市确实是年轻人的聚集地，远的说看b站的up大多搬家后，选择在上海租一间房间，近处身边的同学也在上海有着不小的数量。

再聊聊婚姻打算，婚自然是要结的，压力自是不必说。

两年的时光， 从他刚来用牛角包填肚子，现在有了几道拿手菜，我想这短暂的留学生活，他应该十分充实。

合影后分别，相约上海再见。

我回到了住的地方，房东家人很多，住在附近，恰好这院子比较宽敞，暑假的日子，孩子们聚集在院落中，在运动中释放，欢笑声溢出街道，而老人们用粗糙的手拂过细嫩的树叶，摘下已经结出的果实。

在我看来，人是在跌跌撞撞中继续生活，尽管这路上有很多前人留下的明灯，还是在一不留神中会陷入泥潭，幼年时拥有无惧黑暗的勇气，可能是绊个跟头，也可能落入深渊，随着伤痕增多，开始畏手畏脚，当自己画了一个圈不出来时，只会感叹身边划过的流星是多么的绚烂。

老板的孩子

我的老板是我读博的指导教授，她走起路来风风火火，我们的实验室在走廊的尽头，往往一分钟左右她可以从办公室横穿半个大楼，行事风格干练利落。

感恩节前下午，老板叫我帮她搬一新的显示屏，约在下午一点左右办公室见面，我提前大约五分钟到，正好碰到老板出来，她叫我先等五分钟，不一会她领了女儿出来，大约五岁年龄，可爱伶俐的站在老板腿弯旁，老板和我讲稍等一下，犹豫是把孩子放在实验室让学生带一会还是带她一起去拿显示屏，老板先走向了实验室大约三分钟带着孩子回来了，讲女儿想和她一起去，于是我们出发去图书馆设备处。

老板一马当先，我走在最后，我跟上她的步伐稍显吃力，小姑娘走一会儿自然会距离越来越大，每每这时女孩快跑几步跟上，老板基本不会回头看我们在哪里。在去到老旧的教学楼，需要上一个老式的楼梯，楼梯由单薄的石板构成，两层石板中间可以看到楼下，老板先上了半层，女儿喊道妈妈等等我。老板答道我在这边等你，这个楼梯比较旧你小心一些。孩子扶着和头差不多高的扶手，慢慢向上。

在拿显示屏的地方，是一个室内的开阔二层，楼梯的扶手是一根铁管，在等待的时间，小姑娘跑到楼梯旁的平台眺望，这个平台没有护栏，只是连着楼梯的扶手，当然这个扶手比孩子的身高要高，老板站在孩子的旁边，说你要小心一些之后便静静的看着她。

在回来的路上，孩子帮忙拿了一些小盒子，路上依旧是同样的队形，到了上电梯的时候，老板马上按了向上的按钮，女儿讲道妈妈我想按的，老板说哦，不好意思，我没想到，不过里面也有按钮你可以来按。上了电梯女儿按了三层。

我想这便是平等的教育吧。

幽默与取笑

发表于 2021-04-03 更新于 2023-04-01 分类于 杂文 Waline： 本文字数： 1k 阅读时长 ≈ 1 分钟

幽默与取笑

笑是人类情绪独特的表达，寻找快乐是人的本性，也是人与人快速建立良好关系的方式。本文从个人角度出发，尽量阐述清楚概念理解，了解如何快速建立幽默感。

发笑是因为信息的差异所带来的情感反应。

取笑是从他人缺失的信息出发寻找快乐，这是掌握一定程度幽默感的大部分人的选择，例如，在相声中一般逗哏是取笑角色，而捧哏则是被取笑的一方。这种方式是从伤害一个人而让大家与自身获得快乐，缺陷也十分明显，假如是在生活中这样交流，往往会使被取笑者与取笑者心存芥蒂，因为缺乏对取笑者心理承受能力与双方情谊的考量，把握取笑的程度是困难的，却是十分重要的。

通过取笑而产生的快乐是比较浅层的，通常旁观者的反应是逢场作戏式的迎合，短暂的快乐之后， 会对被取笑者产生一定的同情心，并且降低取笑者的形象。

选择取笑这种方式的原因是，首先表面上来看对自己没有伤害，其次利用自己已知的信息，可以轻易寻找到他人因缺乏对应的信息而做出的行为。

所以取笑的风险比较大，收益比较小，成本不高。

幽默是一种从自己缺失信息出发而寻找快乐的方式，常用自嘲的方式来展示，这种方式是较为低级的幽默，因为听者往往不好把控，选择表现同情还是发笑来进行反馈，毕竟嘲笑有损自身形象。

更为高级的幽默，可以让人发出最为直击心灵的笑声，往往是直接展示自己因为缺失信息而产生的结果与反应，例如卓别林在城市之光中实验的流浪汉，因为对于绅士信息的缺乏，而强行扮演，造成了许多笑料，又比如陈佩斯在主角与配角中，因为缺乏主角正面形象的经验，而强行扮演，也收获的良好的效果，同时在游戏视频中，由于缺乏过关机制的信息，或在博弈中对他人信息的缺失，在探索过程中的失败，也是快乐的源泉。

观众会省去判断与思考直接获取快乐，而在快乐之后的思考也会更加深刻，例如掌握这个信息，从而避免自己犯错，还有同情的心理。

使用伤害自己而产生的幽默，对于本身会造成较大的负荷，同时也不会获取到快乐。

所以幽默的成本高，风险小，收益大。

幽默感的建立，首先是心理承受能力的增强，以抵御伤害自己产生的不良情绪，不过在成功使他人发笑之后会获得成就感，相对抵消。再者把握自己缺乏信息的时刻，最后是表达方式，要使用夸张的方式表现自己的反应，例如夸张的肢体动作，不同的语调，荒诞的逻辑，丰富的面部表情，值得注意的一点是，这个夸张的反应，应当基于当前人物性格，符合事物发展常识。

关于取笑与幽默的选择，人常因为自尊，选择不直接输出结果与反应，而选择自嘲或者隐藏与取笑他人，这样产生的结果往往是对自尊与心灵更为深刻的伤害。

独食的得失

发表于 2020-08-20 更新于 2023-04-01 分类于 散文 Waline： 本文字数： 1.1k 阅读时长 ≈ 1 分钟

独食的得失

一个人吃饭，在当下这个时代或许是一件稀疏平常的事，按照人的时间段来说，最早的时在温暖的子宫，便是一个人汲取母亲的羊水，所幸出生后一段时间是无法自己进食的，同样在这个时间也是急速学习和成长的时候。

长到一定年龄开始有集体生活，托儿所或是幼儿园，义务教育阶段也有寄宿制度，这种阶段虽是一个人吃饭，但往往有同学相伴，只是少了些家人的温暖和随意。

到了大学是个小分水岭，即便选择距离家近的高校，也需在校住宿，而时间没有之前那么固定，事务也相对繁杂，恋爱，社团，兼职等等，这些原因使得有很大概率一个人吃饭，此时也有了很大的选择，可以在食堂躲在人潮中快速了事，可以打包回宿舍看剧享用，可以出门去满足今天的味蕾。

于此同时也有尴尬的地方，吃饭的座位一般设计的是四人桌，最少也是双人座，人少时吃饭是你与老板的博弈，大部分老板是忽视你的存在，这样可以专心享受美食的快乐，但在心理上会涌上一丝寂寞，往往想快速解决以免打扰老板闲暇的生活，有些热情的老板会来搭话，谈的投机吃的也快乐，但也缺乏了独自的感受。

人多的时候就像一滴水进入浪花之中，老板无暇照顾你的感受，毕竟至多只是一个人的收益，拼桌是必须的事实，一个四人桌，拼一对情侣，两人或是三人的朋友，他们无法展开正常的聊天，你也不好贸然搭讪，一顿饭吃的沉默寡言，一个人总是匆匆大败而归。

进入一个人独居的阶段是一个新的领域，这在现代生活中比较常见，在车水马龙中，在高楼林立中安居于半空的一角，这与之前的年代，那个方圆几里的村落，那个大家族式的生活，有了很大的区别，同样对于故乡也少了几分乡土情怀。此时的独食考虑的东西，面临的选择从二维世界变成了三维生活，买什么样的食材，变质的时间，荤素的搭配，口味的选择，质量的大小，这一顿吃什么，下一顿需要怎样的准备，吃还是不吃。

在众多考虑之中，我觉得纠结的问题是这一顿要做多少，一个人吃食量有限，当你洗菜，切菜，热油，挥动五六瓶调味料，然后从锅中舀出一口菜时，未免有些大材小用的怪异感。若是一顿做好几天的菜量，除了第一顿的美味，后面只有消磨的耐心，做一个尽快吃完的剩菜处理机。还有面临饥饿时的烹饪，一边忍耐着肠胃的抗议，一边更加快速的劳动，终于美味呈现在面前的时候，也失去的进食的欲望。同样，为了避免金钱与体力的浪费，自然在买菜时不会只买今天的食材，冷冻食品保存时间可是实在算不上好吃，只是懒惰的福音，新鲜的蔬菜保质期较短，在一顿饭中放多少实在没有准头，多则吃撑，少则丢掉，实在无解。

想想单是吃饭一项，细细想来头疼无比，可这只是生活的一小部分，纵然如此，每次独食也是颇有览阅红尘的快乐。

轻柳

风略黄沙，倒卷起层层散金，夕阳西下，晕染成颗颗红晕。

城镇一角，两双稚嫩的小手合力拔出铁锹，其中一个孩童擦擦额上的汗水，倚靠在墙上，目光热切的看着那个一起种下的胡杨小树苗说道：“小杨柳，以后你就是我小弟了，我罩着你。”

另一个同样目光里掩饰着激动说到：“这是胡杨！什么杨柳。”

“这才显得我们这棵树特殊啊，你懂什么？”

菲丘茑抿抿嘴也不再说，任由吴莫璃意气风发去了。

胡杨，作为边陲小镇常见的植物，默默守护着大漠，正如这个固本镇，作为大皇朝的边界小镇，也在默默守护着王城。

“杨柳”渐渐长大。菲丘茑也愈发出挑了起来，最喜欢的事便是坐在这树下萌荫翻阅手中的古典著作。

吴莫璃跑来，气喘吁吁道：“天天看看看，这到底有什么好看的？快来和我练武！”菲丘茑微微抬头瞟了一眼，叹口气说道：“这江南风景不可亲自去看，在书中体验一番也颇为不错。”

“你倒是体验到了什么好风景说给我听听。”

“哼，你懂什么，知道什么是‘江南多才子’吗？你连对联是什么都不知道，对牛弹琴。“

”不就是对联吗？有我练武有用吗，不和你说了，对琴弹牛。“ 说罢，风风火火带起来一阵尘埃。

菲丘茑轻轻蹙眉又舒展开，继续慢慢抚帙。

时光穿过了流水，仿佛定格。

同样在这“杨柳”下，菲丘茑嘴角上扬，仿佛奇妙之事在手中这方寸之间上演，可惜在这美好时刻又被熟悉的声音打断，

“喂，不就是对联吗？天对地，风对雨。大陆对长空。”被打断的感觉不由得变得有点好笑，视线从书中移开，一位英气的身躯印入眼帘，仔细看去那眼眶倒是青青的，透亮的眼神也微微泛红，想要取笑的话也掩藏起来，

说到：“对的真是妙啊，吴武人今天怎么不去练武？”

吴莫璃不动声色道：“练武总是感觉缺点什么，思来想去缺个人来看我，找你去武场。”菲丘茑合卷起身，

吴莫璃又说道：“之前几日叫你总是不应，怎么今天这么痛快。”

“今天天气好。”

…

“报！外族大举入侵，战事岌岌可危。”

“养兵千日，用兵一时，终日练武只待今日守护国土。“陛下口谕，”征固本镇壮丁，卫大皇朝。“

“杨柳”树下那个挺拔的身影已站了很久，黄沙染上了血的萧飒，菲丘茑看了许久，慢慢走近

“可以不去吗？”

吴莫璃转头凝望，口中坚定的说：“一身武艺保家国，你等我。”

又望向远方缓缓道：“带你去看真正的杨柳。”

菲丘茑定了定神说：“好！”

那天风很大，吹起了红缨。

“前线传来捷报，外族不再进犯！”

看着周围人喜极的表情，菲丘茑舒缓了紧张的心，开始期待再相会的时刻。

“圣上有旨，固本镇抗击有功，升为城池，改为平安城。另纳城主女儿菲丘茑为平安妃，愿世代平安。”

看着周围人庆贺的神情，菲丘茑失去了思考。漠然洗去了刚刚擦好的胭脂，尽管一会儿会有人来画的更好。

就这样身披凤冠霞帔落进了去往皇宫的抬轿，此刻洋洋洒洒飘起了白雪。

“菲丘茑呢？”

“去往皇宫的路上。”

吴莫璃一言不发，跨上了刚下来的战马，一骑绝尘。

这如画的江山裹上了银装，匆匆的马蹄像点点墨梅，为这大好河山挥上了一道不清不黑的浅笔。

白雪马蹄疾，误认终莫离。祈飞知囚鸟，遥盼定情聊。

烟雨的江南，你在为谁哭泣？一叶扁舟上，两岸杨柳依依。

南雪

南方的雪，与北方截然不同。

从物理角度来讲，水只有三种形态，气液固，雪自然属于那固态，科学就像一个霸道的人，粗莽的将事物划开，一如开天的盘古，但盘古劈开的并不是简单的天地，而是将一片混沌化为万物生机。先来谈谈北方的雪，北，深入内陆，降水少，冬天又寒，则感到十分干冷，雪的次数不多，恰如“物以稀为贵”，人走在街上心中自然情不自禁叹道快下场雪吧，北方人见雪开心但不同与南方人，他们看雪像是一位老朋友，一年也不会多见，每见必会热情相待。降雪时少风，想起旧时比喻“鹅毛般的大雪”并不是徒有虚词，雪飘着很慢，轻轻落在地上，周围世界很静，可以听到气流通过身体的嘶嘶声，那夹着湿意空气润浸了肺，不忍呼出怕搅乱自由落下的雪花，却忍不住大口吸入似久旱逢甘霖。当开始下雪后，这位老朋友便止不住他的才情，洋洋洒洒将大地铺上一层厚毯。用手抓一把，明明是水但干的嘎吱嘎吱作响。

再来谈谈南方雪，来这边听闻会下雪，生活过的两年大学生活并未有幸见到，今年寒潮期预计这边会有降雪，心中不免有些期待，接近下雪的日子开始无休止的下雨，一天未停，一夜未停，早晨打伞出门落于伞上的声音成了类似小纸蛋打在布片那样，地面上全是积水，渐渐地有了白色，那极速降落的东西不愿称之为雪，就如同小冰粒在坠落，打在脸上生疼，疾风裹挟着冰粒砸在路上，没有兴奋，堆积的雪踩着一声脆响，虽是大家都不曾见过的风景，我选择走过，有人选择落墨，看到草坪上摆着大大小小的雪人，有趣。新奇南方的雪，更醉心北方雪。

在家时常嫌雪小，远方时长叹雪少，见面时浪念雪糟。习惯于鹅毛漫天，习惯于盈满明圆，习惯于美满合欢，殊不知四季交替，常缺细剔，远赴及第是常态，失去的空虚妄图拿新奇填满，仿若跌入黑洞难见光明。怕习惯于习惯。

冬天来了，听到了春的呼唤。
